Program diatas merupakan program dari fungsi def untuk menampilkan data mahasiswa dengan nilai yang diinput.

serta menggunakan fungsi if statement untuk melakukan pemanggilan ke fungsi def. 
Juga menggunakan fungsi dictionary untuk mengumpulkan data dari nama dan nilai yang diinput user.
